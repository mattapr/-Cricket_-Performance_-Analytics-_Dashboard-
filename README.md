# 🏏 Cricket Performance Analytics Dashboard

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** built using international cricket match data. It provides comprehensive batting and bowling performance analysis for selected Indian cricket players through dynamic visualizations, KPI cards, and interactive slicers.

The dashboard enables users to explore player performance across different years and match formats (Test, ODI, and T20). It combines data modeling, DAX calculations, and Power BI visualizations to transform raw cricket statistics into meaningful insights for performance analysis and decision-making.

---

## 📂 Dataset

The project uses three primary datasets:

* **bat.csv** – Batting statistics
* **ball.csv** – Bowling statistics
* **match.csv** – Match information

The datasets were imported into Power BI and modeled using relationships between player IDs, scorecard IDs, and match dates.
